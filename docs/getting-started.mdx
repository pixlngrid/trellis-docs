Welcome to **Trellis** — an opinionated docs framework built on Docusaurus with custom theme enhancements, smart search, and a design token system.

## What's Included

| Category | Highlights | Learn More |
|----------|-----------|------------|
| **Theme** | Last-updated at top, heading copy-to-clipboard, pill-style tabs, custom admonition icons | [Theme Overview](/theme/) |
| **Design Tokens** | JSON-to-CSS pipeline for centralized brand management | [Design Tokens](/design-tokens/) |
| **Plugins** | Smart search, FAQ indexer, redirects, image lightbox, Mermaid pan/zoom | [Plugins Overview](/plugins/) |
| **Components** | Glossary, feedback widget, flipping cards, search UI | [Components](/components/) |

New to Trellis? Read [What is Trellis?](/overview/) for the full picture, or see the [feature comparison](/overview/trellis-vs-docusaurus/) with vanilla Docusaurus.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/pixlngrid/trellis.git
cd trellis

# Install dependencies
yarn install

# Start the dev server
yarn start
```

:::tip
The dev server runs on port 3001 by default. Visit `http://localhost:3001` to see your docs.
:::

## Project Structure

```
trellis/
├── docs/              # Your documentation content
├── blog/              # Release notes and blog posts
├── src/
│   ├── css/           # Custom CSS and design tokens
│   ├── theme/         # Swizzled Docusaurus theme components
│   ├── components/    # Reusable React components
│   └── pages/         # Custom pages
├── static/            # Static assets (images, fonts)
├── packages/          # Local plugins
│   ├── redirects-plugin/
│   └── faq-index-plugin/
├── design-tokens.json # Design token definitions
└── docusaurus.config.js
```
