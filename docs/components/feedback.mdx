The Feedback component provides an embeddable widget for collecting page-level feedback from readers.

## Usage

```mdx title="Any .mdx file"
import Feedback from '@site/src/components/Feedback/Feedback';

<Feedback />
```

The widget renders a "Was this page helpful?" prompt with thumbs up/down buttons and an optional text input for detailed feedback.

## How It Works

1. The user clicks a thumbs up or thumbs down button
2. An optional text field appears for additional comments
3. On submit, the feedback is sent to the configured API endpoint
4. A confirmation message is shown

## API Configuration

The feedback API endpoint is configured in `src/components/Feedback/api.js`:

```js title="src/components/Feedback/api.js"
// Development: http://localhost:3002/api/feedback
// Production: /api/feedback
```

To use the feedback widget in production, you need a backend endpoint that accepts POST requests with the following payload:

```json
{
  "page": "/current-page-path/",
  "helpful": true,
  "comment": "Optional user comment"
}
```

## Feedback Dashboard

A companion `FeedbackDashboard` component is included at `src/components/Feedback/FeedbackDashboard.js` for viewing collected feedback. This is intended for admin/internal use.

## Customization

The widget's "Report an issue" link uses the repository URL from `siteConfig.customFields.repoUrl`. Update this in `docusaurus.config.js`:

```js title="docusaurus.config.js"
customFields: {
  repoUrl: 'https://github.com/your-org/your-repo',
},
```

The widget styling uses the brand primary color from design tokens. To adjust the appearance, modify `src/components/Feedback/Feedback.js`.
