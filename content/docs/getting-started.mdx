## Prerequisites

Before getting started, make sure you have the following installed:

- **Node.js 18+** — [download here](https://nodejs.org/)
- **npm**, **yarn**, or **pnpm** — npm comes bundled with Node.js
- **Git** — [download here](https://git-scm.com/)

You can verify your setup by running:

```bash
node --version   # Should print v18.x or higher
npm --version    # Should print 9.x or higher
git --version
```

## Quick Start

### Option A: Scaffold a new project

The fastest way to start a new docs site:

```bash
npx create-trellis-docs my-docs
cd my-docs
npm run dev
```

You'll be prompted for a site title, tagline, URL, and optional GitHub repo URL.

### Option B: Clone the repo

```bash
git clone https://github.com/pixlngrid/trellis.git
cd trellis

npm install
npm run dev
```

:::tip
The dev server runs on port 3000 by default. Visit `http://localhost:3000` to see your docs.
:::

```
├── scripts/               # Build scripts (tokens, search index, FAQ index)
├── design-tokens.json     # Design token definitions
└── package.json
```
