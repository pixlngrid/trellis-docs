{vars.productName} bundles five plugins that extend the framework with search, FAQ indexing, redirects, image zoom, and diagram pan/zoom.

## Bundled Plugins

| Plugin | Type | Description |
|--------|------|-------------|
| [Smart Search](/plugins/smart-search/) | Build-time + client | Full-text search with Fuse.js — no external service required |
| [FAQ Index](/plugins/faq-index/) | Build-time | Auto-generates a searchable FAQ table of contents from `###` headings |
| [Redirects](/plugins/redirects/) | Build-time | Manages URL redirects via JSON — generates HTML meta-refresh pages |
| Image Lightbox | Client-side | Click-to-zoom on any markdown image — no markup changes needed |
| Mermaid Pan/Zoom | Client-side | Pan and zoom on Mermaid diagrams |

## Plugin Registration

All plugins are registered in `config/site.ts`:

```ts title="config/site.ts"
plugins: [
  'panzoom-plugin',
  ['./packages/faq-index-plugin', {
    faqDir: 'content/docs/faq',
    basePermalink: '/faq',
  }],
  ['redirects-plugin', {
    redirectsFile: 'redirects.json',
  }],
  ['smart-search-plugin', {
    excludedFolders: ['includes', '_includes'],
    excludedPrefixes: ['_'],
    searchWeights: {
      title: 1.0,
      'sections.heading': 1.0,
      keywords: 0.8,
      description: 0.6,
      'sections.content': 0.5,
      content: 0.4,
    },
  }],
  ['lightbox-image-plugin', {
    selector: '.markdown img',
    background: 'rgba(0, 0, 0, 0.8)',
    zIndex: 999,
  }],
],
```

## Image Lightbox

The lightbox plugin automatically makes all images in markdown content zoomable. Click any image to open it in a full-screen overlay. No special syntax needed — standard markdown images work:

```markdown
![Screenshot of the dashboard](./images/dashboard.png)
```

### Configuration

```js
['lightbox-image-plugin', {
  selector: '.markdown img',    // CSS selector for images to enhance
  background: 'rgba(0, 0, 0, 0.8)',  // Overlay background color
  zIndex: 999,                  // CSS z-index for the overlay
  margin: 10,                   // Margin around the zoomed image (px)
  scrollOffset: 10,             // Scroll offset before closing
}]
```

## Mermaid Pan/Zoom

Mermaid diagrams are rendered with built-in Mermaid support and enhanced with pan/zoom. This is configured in the theme config:

```ts title="config/site.ts"
themeConfig: {
  zoom: {
    selectors: ['div.mermaid[data-processed="true"]'],
    enableWheelZoom: true,
    enableDoubleClickResetZoom: true,
  },
},
```

Users can scroll-wheel to zoom in/out and click-drag to pan. Double-click resets to the original view.
